## Run Locally

Clone the project

```bash
  git clone https://github.com/meczajkowski/Invo-Spotify-part-6.git
```

Go to the project directory

```bash
  cd Invo-Spotify-part-6
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
